# YOLOv8-bicycle-helmet-detection

Graduation project repository for bicycle helmet detection based on YOLOv8.

Included in this repository:
- Source code for the base YOLOv8 project
- Source code for the YOLOv8 + CBAM variant
- UI resources, model weights, test materials, and project documents

Excluded from git:
- Training datasets
- Training run outputs
- Generated save results
- Archive and temporary files
